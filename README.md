# LINKUP - Profile Settings Page

## Description
LINKUP is a modern social platform with a clean and customizable UI built on Laravel 12. This project demonstrates:
- User authentication (login/register)
- Profile management
- Sidebar & header layout reuse
- Tailwind CSS custom theme

## Features
- Responsive sidebar & header
- Glass-card UI elements
- Profile editing (username, display name, bio)
- Dark mode support
- Tailwind CSS 3.x with plugins (forms, container-queries)
